Проверка на пустые массивы

Иногда случается, что нужно проверить, являются ли массивы пустыми одновременно. Проверка с использованием foobar.length == 0 выглядит слишком длинной. Заменим её простым сложением массивов. Да-да, сложением:
var a = [...], b = [...], ... z = [...]; a + b + ... + z == ''; // true
